# Inventario de documentos detectados

!!! warning "Aviso"
    LEY.DO no es una fuente oficial. Este inventario no certifica exhaustividad, vigencia ni validez legal. Los documentos listados están pendientes de descarga, hash, normalización y revisión humana.

## Qué es esta sección

Esta sección muestra documentos legales dominicanos detectados en fuentes oficiales durante la fase de inventario. No equivale todavía a publicación de cada documento como texto normalizado de LEY.DO.

Para que un documento pase del inventario al corpus normalizado de LEY.DO debe tener, cuando aplique: PDF oficial preservado, hash SHA256, Markdown, JSON de metadata y revisión humana.

## Fuentes oficiales contrastadas

- Presidencia de la República Dominicana: <https://presidencia.gob.do/decretos> y <https://presidencia.gob.do/leyes>
- Consultoría Jurídica del Poder Ejecutivo: <https://www.consultoria.gov.do/consulta/>

## Lectura recomendada

- Use las páginas por año para revisar documentos detectados.
- Los enlaces de Consultoría son enlaces oficiales de consulta o descarga, pero no se han verificado todavía como PDF original en esta fase.
- La ausencia de un documento en este inventario no prueba inexistencia.

## Archivos fuente del inventario

Los archivos JSON/Markdown técnicos quedan en el repositorio bajo `fuentes/` para auditoría y procesamiento posterior.
